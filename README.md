# C_Loader
cs loader，bypassav， 分离免杀某60，某绒等国内主流杀软

go run ./encoder.go生成临时文件，然后将编译main.c生成的loader与生成的临时文件放在同一目录下。
在没有其他文件干扰的情况（to fix）下可以正常上线。
